# Dashboard de Vendas - Xbox

## 📌 Objetivo
Transformar dados brutos de assinaturas em um **dashboard interativo no Excel**, permitindo análise clara do desempenho de vendas e suporte à tomada de decisão.
## 🚀 Como reproduzir
1. Abra o arquivo `base.xlsx` no Excel.
2. Importe os dados para uma **Tabela Dinâmica**.
3. Crie os **KPIs** usando fórmulas simples (`SOMA`, `MÉDIA`, `CONT.SE`).
4. Monte os gráficos recomendados a partir da tabela dinâmica.
5. Organize os KPIs no topo e os gráficos em seções laterais para formar o dashboard.

## ✅ Resultado
Um dashboard visual e interativo que permite:
- Identificar quais planos e benefícios geram mais receita.
- Acompanhar a evolução das assinaturas ao longo do tempo.
